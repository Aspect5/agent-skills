# Guardrail report template

The deliverable shape. Write the full report to `guardrail-<surface>.md` (repo root
or a user-named path) **incrementally** as each guardrail is built, and emit only a
tight inline summary: the surface(s) blocked, where each is wired, and the two test
results per guardrail. Never paste a discovered secret value into the report —
reference it by name, mark it redacted.

> Only load this file when formatting the report.
